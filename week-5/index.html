CSS Flexbox

Flexbox

Flexbox is a one-dimensional layout method for arranging items in rows or columns. Items flex (expand) to fill additional space or shrink to fit into smaller spaces.

The main idea behind the flex layout is to give the container the ability to alter its items’ width/height (and order) to best fill the available space (mostly to accommodate to all kind of display devices and screen sizes).

Some of flex properties are meant to be set on the container (parent element, known as “flex container”) whereas the others are meant to be set on the children (said “flex items”). The properties for the parent element (the flex container) and the child elements (the flex items).

Note: Flexbox layout is most appropriate to the components of an application, and small-scale layouts, while the Grid layout is intended for larger scale layouts.

Common flexbox Properties

  • display: flex = Makes an element a flex container and its children to become flex items.
  • flex-direction: row(default)/column/row-reverse/column-reverse = that specifies which direction the main axis runs (which direction the flexbox children are laid out in).
  • flex-wrap: wrap = makes elements wrap down/go down the layout when space is finished in the line they are placed in
  • flex: A shorthand property that can specify up to three different values: flex-grow , flex-shrink, flex-basis
  • justify-content: flex-start/flex-end/center/space-between/space-around/ = This controls where the flex items sit on the main axis.
  • align-items:flex-start/flex-end/center/stretch = This controls where the flex items sit on the cross axis. You can override the align-items behavior for individual flex items by applying the align-self property to them.
  • order: Used for changing the layout order of flex items without affecting the source order NOTE: By default, all flex items have an order value of 0. Flex items with higher specified order values will appear later in the display order than items with lower order values.

--- parent element (the flex container) - display: flex - flex-direction - flex-wrap - flex-flow (flex-dorection and flex-wrap shorthand) - justify-content: flex-start/flex-end/center/space-between/space-around/stretch - align-items:flex-start/flex-end/center/stretch - align-content: flex-start/flex-end/center/space-between/space-around/stretch NOTE on align-content: This aligns a flex container’s lines within when there is extra space in the cross-axis, similar to how justify-content aligns individual items within the main-axis. This property only takes effect on multi-line flexible containers, where flex-wrap is set to either wrap or wrap-reverse)

--- child elements (the flex items) - flex-grow - flex-shrink - flex-basis - flex: A shorthand property that can specify up to three different values: flex-grow , flex-shrink, flex-basis - align-self: overrides the align-items behavior for individual flex items by applying the align-self property to flex item - order: Used for changing the layout order of flex items without affecting the source order NOTE: By default, all flex items have an order value of 0. Flex items with higher specified order values will appear later in the display order than items with lower order values.
