Custom Properties and Functions

Custom Properties

Custom properties (sometimes referred to as CSS variables or cascading variables) are entities defined by CSS authors that contain specific values to be reused throughout a document. They are set using custom property notation (e.g., --main-color: black;) and are accessed using the var() function (e.g., color: var(--main-color);).
CSS variables are custom properties that cascade normally and inherit. Custom properties allow a value to be stored in one place, then referenced in multiple other places.
An additional benefit is semantic identifiers. For example, --main-text-color is easier to understand than #00ff00, especially if this same color is also used in other contexts.

Custom properties in CSS are used for for tokenising your CSS, but also for abstracting complexity and interactivity into smaller, easier to manage pieces. Note: Variables do not work inside media queries and container queries. When CSS variables are set externally (e.g., through HTML or JavaScript), use them for pure data, not CSS values like lengths or percentages.

Variable Declaration and Use

Declaring a custom property is done using a custom property name that begins with a double hyphen (--), and a property value that can be any valid CSS value.
element { --main-bg-color: brown; }
A common best practice is to define custom properties on the :root pseudo-class, so that it can be applied globally across your HTML document:
:root { --main-bg-color: brown; } Custom property names are case sensitive â€” --my-color will be treated as a separate custom property to --My-color.

CSS Comparison Functions: min(), max(), and clamp()

CSS Comparison Functions are about comparing multiple values and representing one of them based on the function used. In min() and ma() functions, the position where min or max value is placed does not matter, the min or max value will be selected according to the function used, min() or max().

The min() Function

The min() function contains one or more comma-separated calculations and represents the smallest value of them. We use the min() to set a maximum value. Example
.element { width: min(50%, 500px); }

The max() Function

The max() function contains one or more comma-separated calculations and represents the largest value of them. We use the max() to set a minimum value.

The Clamp() Function

What clamp() do is that it clamps a value between two defined values, minimum and maximum. It takes three parameters (min value, preferred value, max value). Example
.element { width: clamp(200px, 50%, 1000px); }
