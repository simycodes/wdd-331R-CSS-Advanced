CSS Grid

Grid

Grid layout is used to align elements into columns and rows. A CSS Grid is used to divide a web page or container into columns and rows, along with the ability to position and size the building block elements based on the rows and columns we have created in terms of size, position, and layer. The grid also gives us a flexible way to change the position of elements with only CSS without any change to the HTML. This can be used with media queries to alter the layout at different breakpoints.

Common Grid Properties

  • grid: Makes an element a grid container (display: grid)
  • grid-template-columns: Creates columns of the grid with a space-separated list of values.
  • grid-template-rows: Defines rows of the grid with a space-separated list of values.
  • grid-auto-columns: Specifies the size of any auto-generated grid columns-tracks
  • grid-auto-rows: Specifies the size of any auto-generated grid row-tracks
  • grid-column-gap: Creates a ga between the columns
  • grid-row-gap: Creates a ga between the rows
  • grid-gap (short-hand for grid-column-gap and grid-row-gap)

Parent (specific) Grid Properties

  • justify-items: Aligns grid items along the inline (row) axis - start/end/center/stretch
  • align-items: Aligns grid items along the block (column) axis - start/end/center/stretch(default)/baseline

  • justify-content: This property aligns the grid along the inline (row) axis. start/end/center/stretch/space-around/space-between-space-evenly
  • align-content: This property aligns the grid along the block (column) axis. start/end/center/stretch/space-around/space-between-space-evenly
  • place-content: place-content sets both the align-content and justify-content properties in a single declaration.

Grid cells/children specific Properties

  • grid-column-start
  • grid-column-end
  • grid-column (short-hand for grid-column-start and grid-column-end)
  • row-column-start
  • row-column-end
  • grid-row (short-hand for grid-row-start and grid-row-end)
  • justify-self: Aligns a grid item inside a cell along the inline (row) axisb - start | end | center | stretch;
  • align-self: Aligns a grid item inside a cell along the block (column) axis - start | end | center | stretch;
  • place-self: place-self sets both the align-self and justify-self properties in a single declaration.
